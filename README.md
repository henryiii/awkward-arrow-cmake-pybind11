# awkward-arrow-cmake-pybind11

## Getting Started

Remember to use

```console
git clone --recursive
```

to get the `pybind11` submodule.

## Building and Debugging

```
make build-image-debug
make debug
bash build.sh
```

## Run examples after building

```
bash run_example.sh
```
